ACDSKBackgroundSessionEventHandler Class Reference

Inherits from NSObject
Declared in ACDSKBackgroundSessionEventHandler.h

Overview

The object that can be used to re-register action blocks to handle error, progress and success for background request such as ACDSKDownloadFileRequest after application is terminated.

It is recommended to use as soon as possible either in application:handleEventsForBackgroundURLSession:completionHandler: or -application:didFinishLaunchingWithOptions: if launchOptions does not contain “UIApplicationLaunchOptionsURLSessionKey” key.
